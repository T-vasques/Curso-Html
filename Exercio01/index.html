Comunicação Cliente-Servidor

Cliente:

O cliente é um dispositivo (como um computador, smartphone ou tablet) ou um programa de software que solicita serviços ou recursos de outro dispositivo ou programa.O cliente envia solicitações para um servidor, pedindo informações, serviços ou recursos específicos. Ao abrir um navegador da web e digitar um endereço (URL), você está atuando como um cliente. O navegador solicita páginas da web a um servidor para exibi-las. Servidor:

O servidor é um dispositivo ou software que fornece serviços, recursos ou informações solicitadas pelo cliente.Recebe as solicitações do cliente, processa essas solicitações e envia de volta as informações ou recursos necessários. Exemplo,Um servidor de e-mail recebe, armazena e envia e-mails para os clientes quando são solicitados.

Interação Cliente-Servidor:

O cliente e o servidor se comunicam por meio de protocolos, como HTTP para páginas da web ou SMTP para e-mails. O cliente envia uma solicitação específica ao servidor, que processa essa solicitação e retorna uma resposta adequada ao cliente. Essa comunicação pode ser estabelecida por redes, como a internet, onde o cliente e o servidor podem estar em locais diferentes, mas conectados por meio de uma rede.

O que é o HTML ?

O HTML (Hypertext Markup Language) é a linguagem de marcação fundamental para criar páginas da web. Ele funciona como a estrutura esquelética dessas páginas, definindo como o conteúdo deve ser organizado e exibido. Em vez de ser uma linguagem de programação, o HTML é uma linguagem de marcação, usando tags para identificar diferentes elementos, como títulos, parágrafos, imagens e links. Essas tags são interpretadas pelos navegadores da web, permitindo que eles exibam o conteúdo conforme as instruções do código HTML. Em resumo, o HTML é a base sobre a qual toda a estrutura das páginas da web é construída, determinando como o conteúdo é apresentado aos usuários.

Alguns exemplos de tags:

  • h1, h2, h3, ... h6: define títulos de diferentes níveis de importância.
  • p : cria parágrafo
  • a : cria link
  • img : insere uma imagem
  • div e span : são contêineres genéricos usados para agrupar outros elementos e aplicar estilos
